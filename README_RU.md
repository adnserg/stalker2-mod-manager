# Менеджер модов S.T.A.L.K.E.R. 2

Приложение для управления модами S.T.A.L.K.E.R. 2: Heart of Chornobyl.

## Возможности

1. **Загрузка модов** - Загрузка списка модов из папки Vortex
2. **Сортировка модов** - Изменение порядка модов (влияет на префиксы папок: AAA-, AAB-, AAC-, и т.д.)
3. **Сохранение/Загрузка конфигурации** - Сохранение текущего списка модов и их настроек (отдельные файлы для путей и порядка модов)
4. **Включение/Выключение модов** - Управление тем, какие моды будут установлены
5. **Установка модов** - Автоматическое создание папок с префиксами и копирование файлов модов
6. **Перетаскивание для изменения порядка** - Визуальное перетаскивание модов для изменения их порядка в списке
7. **Экспорт/Импорт порядка модов** - Обмен порядком модов с другими пользователями
8. **Продвинутая сортировка** - Сортировка модов согласно внешним JSON/TXT файлам (например, снапшот Vortex)
9. **Умная установка** - Копирует только измененные или отсутствующие файлы, пропуская идентичные
10. **Сохранение размера окна** - Запоминает размер окна между сеансами
11. **Логирование** - Подробное логирование в файл `app.log`
12. **Поддержка нескольких языков** - Переключение между английским и русским

## Использование

1. **Укажите пути:**
   - **Путь к Vortex** - Путь к папке с модами из Vortex (например: `E:\Vortex Mods\stalker2heartofchornobyl`)
   - **Путь к ~mods** - Путь к папке `~mods` (например: `E:\SteamLibrary\steamapps\common\S.T.A.L.K.E.R. 2 Heart of Chornobyl\Stalker2\Content\Paks\~mods`)

2. **Загрузите моды:**
   - Нажмите "Загрузить моды" для загрузки списка модов из папки Vortex

3. **Настройте моды:**
   - Используйте чекбоксы для включения/выключения модов
   - Перетаскивайте моды в списке для изменения их порядка
   - Или используйте кнопки "Вверх" и "Вниз" для изменения порядка модов

4. **Сохраните конфигурацию:**
   - Нажмите "Сохранить конфиг" для сохранения текущих настроек (пути и порядок модов сохраняются отдельно)

5. **Экспорт/Импорт порядка модов:**
   - Нажмите "Экспорт порядка" для сохранения порядка модов в JSON файл
   - Нажмите "Импорт порядка" для загрузки порядка модов из JSON файла

6. **Дополнительные настройки:**
   - Нажмите "Дополнительно" для открытия окна дополнительных настроек
   - Включите "Сортировать моды согласно файлу JSON/TXT"
   - Выберите JSON или TXT файл (например, снапшот Vortex snapshot.json или файл с пользовательским порядком)
   - Порядок модов будет отсортирован согласно файлу

7. **Установите моды:**
   - Нажмите "Установить моды" для установки всех включенных модов в папку ~mods
   - Приложение будет:
     - Проверять каждый файл перед копированием
     - Пропускать файлы, которые уже идентичны (по размеру и дате изменения)
     - Копировать только измененные или отсутствующие файлы
     - Удалять неиспользуемые папки модов

## Формат папок

Моды устанавливаются в папки с префиксами:
- `AAA-ModName` (первый мод)
- `AAB-ModName` (второй мод)
- `AAC-ModName` (третий мод)
- и т.д.

Порядок префиксов: AAA, AAB, AAC, ..., AAZ, ABA, ABB, ..., ABZ, ACA, и т.д.

## Файлы конфигурации

Приложение использует два отдельных файла конфигурации:
- `paths_config.json` - Хранит путь к Vortex, путь к цели, размер окна и язык
- `mods_order.json` - Хранит порядок модов и статус включения (может быть экспортирован/импортирован)

## Поддержка языков

Приложение поддерживает английский и русский языки. Вы можете переключать языки с помощью кнопки в заголовке окна.

## Требования

- .NET 9.0 Windows Runtime
- Windows 10/11

## Сборка

```bash
dotnet build
```

## Запуск

```bash
dotnet run
```

Или запустите скомпилированный `.exe` файл из папки `bin/Debug/net9.0-windows/`.

## Логирование

Все действия приложения записываются в файл `app.log` в директории приложения. Уровни логов:
- Debug - Детальная информация для отладки
- Info - Общая информация
- Success - Успешные операции
- Warning - Предупреждения
- Error - Ошибки
