# S.T.A.L.K.E.R. 2 Mod Manager

Приложение для управления модами S.T.A.L.K.E.R. 2: Heart of Chornobyl.

## Возможности

1. **Загрузка модов** - Загрузка списка модов из папки Vortex
2. **Сортировка** - Изменение порядка модов (влияет на префиксы папок: AAA-, AAB-, AAC-, и т.д.)
3. **Сохранение/Загрузка конфигурации** - Сохранение текущего списка модов и их настроек
4. **Включение/Выключение модов** - Управление тем, какие моды будут установлены
5. **Установка модов** - Автоматическое создание папок с префиксами и копирование файлов модов

## Использование

1. **Укажите пути:**
   - **Vortex Path** - путь к папке с модами из Vortex (например: `E:\Vortex Mods\stalker2heartofchornobyl`)
   - **Target Path** - путь к папке `~mods` (например: `E:\SteamLibrary\steamapps\common\S.T.A.L.K.E.R. 2 Heart of Chornobyl\Stalker2\Content\Paks\~mods`)

2. **Загрузите моды:**
   - Нажмите "Load Mods" для загрузки списка модов из папки Vortex

3. **Настройте моды:**
   - Используйте чекбоксы для включения/выключения модов
   - Используйте кнопки "Move Up" и "Move Down" для изменения порядка модов

4. **Сохраните конфигурацию:**
   - Нажмите "Save Config" для сохранения текущих настроек

5. **Установите моды:**
   - Нажмите "Install Mods" для установки всех включенных модов в папку ~mods
   - Внимание: все существующие файлы в папке ~mods будут удалены!

## Формат папок

Моды устанавливаются в папки с префиксами:
- `AAA-ModName` (первый мод)
- `AAB-ModName` (второй мод)
- `AAC-ModName` (третий мод)
- и т.д.

Порядок префиксов: AAA, AAB, AAC, ..., AAZ, ABA, ABB, ..., ABZ, ACA, и т.д.

## Сборка

```bash
dotnet build
```

## Запуск

```bash
dotnet run
```

Или запустите скомпилированный `.exe` файл из папки `bin/Debug/net8.0-windows/`
