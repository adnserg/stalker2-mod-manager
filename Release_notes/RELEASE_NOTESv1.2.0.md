# Release Notes - v1.2.0

## S.T.A.L.K.E.R. 2 Mod Manager - Update Release

### Major Changes

#### New Features

##### Hot Keys
- **Hot Key Support** - Added hot key support for quick access to functions
- **Faster Workflow** - Use keyboard shortcuts for more efficient work

##### Mod List Change Tracking
- **Save/Cancel Buttons** - Added buttons to save or cancel changes in the mod list
- **Change Tracking** - The application tracks changes to mod order and their state (enabled/disabled)
- **Close Warning** - A warning appears when closing the window with unsaved changes
- **Improved Checkbox Handling** - Ctrl+click for multi-select, batch state changes
- **Auto-load Mods** - Mods are automatically loaded on startup if the folder path is configured
- **Auto-load Settings** - Settings are automatically loaded on application start

##### Mod Sorting
- **Mod Sorting** - Added ability to sort the mod list
- **Sort by State** - Sort mods by enabled/disabled

##### Launch Button
- **Game Launch** - Added button for quick game launch
- **Convenient Access** - Launch the game directly from the mod manager

##### Improved Drag & Drop
- **File Dragging** - Improved drag and drop functionality for mod files
- **More Stable Operation** - Fixed issues with drag and drop event handling

##### Path Saving
- **Auto-save Paths** - Paths are now automatically saved
- **Improved Validation** - Enhanced path validation

### Technical Improvements

- **Improved Localization** - Added localization for warnings and save/cancel buttons
- **Design Fixes** - Various UI and style improvements
- **Bug Fixes** - Fixed various issues and errors
- **Improved Warnings** - Enhanced warning windows and their functionality

### Installation

1. Download the v1.2.0 release from GitHub
2. Extract the application
3. Run `Stalker2ModManager.exe`
4. Enjoy the new features!
